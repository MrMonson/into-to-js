DOM: Targeting

To get access to an Element(object) the document gives us a few options

document.getElementsByTagName()

params: this will take a string of the tag you want to target

'div' will get an array like object full of all elements that are div's

document.getElementsByTagName('p')[0]

will give you access to the first p tag on the page

document.getElementsByClassName()

params: this will take a string of the class you want to target

'item' will get an array like object full of all elements with a class of item's

document.getElementsByClassName('item')[1]

will give you access to the second element with the class item on the page

document.getElementByID()

params: this will take a string of the id you want to target

unlike the methods that start document.getElements

this one is singular document.getElement

Instead of an array of dom nodes this will be a dom node
