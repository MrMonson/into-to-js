GIT

git add < file name >

will add a single file

git add .

will add all files

git commit

will bring up a vim console to input a message

git commit -m "your message"

will bypass the vim console and use your messages

git push

will push your committed work to the branch you are on

git push -u yourBranch

will push your committed work on your local branch when the upstream has not been set

git checkout yourBranch

will move you to the branch you want to switch to

git checkout -b yourBranch

will create a new branch based off your current branch and switch you to it
