DOM: Creating Elements

document.creatElement( string )

params: this will take a string. The tag you want to create

'div' Will return a div element node.

Note: this will not be enough to view an element

parentNode.appendChild(childNode)

params: this will take a nodeElement that you want to become the childNode of your parentNode

const ul = document.createElement('ul');
document.body.appendChild( ul );

will create an ul and append it to the body of an html page

const li = document.createElement('li');
document.getElementsByTagName('ul')[1].appendChild( li );

will create an li and append it to the second ul on an html page
